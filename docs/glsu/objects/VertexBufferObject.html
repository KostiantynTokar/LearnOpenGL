    Alias VertexBufferObject

    Represents raw data in GPU memory.

    alias VertexBufferObject = BufferObejct!(34962);

    See Also

    BufferObject for VertexBufferObject methods documentation.

    Example

    // Buffer in main memory.
float[] abstractData = [
    -0.5f, -0.5f,
     0.5f, -0.5f,
     0.0f,  0.5f,
];
// Transfer data to GPU memory.
auto VBO = VertexBufferObject(abstractData, DataUsage.staticDraw);
scope(exit) VBO.destroy();
