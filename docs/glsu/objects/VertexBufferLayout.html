    Struct VertexBufferLayout

    Represents VertexBufferObject layout. Works as an array of AttribPointer's.

    struct VertexBufferLayout ;

    Allows specifying a layout step-by-step. Calls to push and pushUsingPattern should be done accordingly to the offset of vertex attributes, so that attributes with lesser offset should be pushed earlier.
