    Template unpack

    Tuple unpacker

    template unpack(alias func) ;

    Contained Functions

    NameDescription
    unpack

    Usage

    myTuple.unpack!((x, y) => f(x, y));

    Arguments are bound by order; names are irrelevant.

    See Also

    Based on tupArg by Dukc: An (old/new?) pattern to utilize phobos better with @nogc. packWith.

    Example

    import std.range : zip, repeat;
import std.algorithm : filter, map, each;

const int j = 2;
int i = 0;
const int[3] tmp = [1, 2, 3];

// tmp[]
//     .filter!((x)scope => x == j) // lambda closes over variable j
//     .each!((x)scope => i = x);
tmp[]
	.zip(repeat(j))
	.filter!(unpack!((x, j) => x == j))
    .map!(unpack!((x, j) => x))
    .each!((x) scope => i = x);

assert(i == 2);
